\# 🛒 Smart Cart - Carrinho Inteligente para Supermercados



Sistema de carrinho de compras inteligente que permite ao cliente escanear produtos, confirmar e pagar diretamente pelo carrinho, sem filas!



\## 🚀 Demonstração



\- \*\*Carrinho:\*\* \[smart-cart.vercel.app](https://smart-cart.vercel.app)

\- \*\*App do Mercado:\*\* \[smart-cart-mercado.vercel.app](https://smart-cart-mercado.vercel.app)



\## 📱 Funcionalidades



\### Smart Cart (Carrinho)

\- ✅ Login via QR Code

\- ✅ Leitura de código de barras (câmera ou manual)

\- ✅ Sistema de segurança com confirmação em 5 segundos

\- ✅ Lista de produtos em tempo real

\- ✅ Pagamento PIX (QR Code)

\- ✅ Pagamento NFC/Aproximação (simulado)

\- ✅ Interface responsiva para tablet/celular



\### App do Mercado

\- ✅ Geração de QR Code para login

\- ✅ Visualização de carrinhos ativos

\- ✅ Simulação de movimentação em tempo real



\## 🛠️ Tecnologias Utilizadas



\- \*\*Frontend:\*\* HTML5, CSS3, JavaScript (Vanilla)

\- \*\*Backend:\*\* Firebase Realtime Database

\- \*\*Hospedagem:\*\* Vercel

\- \*\*Controle de Versão:\*\* GitHub
